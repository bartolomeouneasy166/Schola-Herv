## ⚙️ How It Works

Schola-Herv uses an asynchronous engine to retrieve data. This means it requests multiple files at the same time to increase speed. The tool checks for duplicates so you do not download the same paper twice. 

The application saves a record of your searches in a database. This allows you to track your progress and resume a download session later. You can modify the file destination path in the settings menu.

## ❓ Frequently Asked Questions

**Does the software store my credentials?**
The software does not require a login. It pulls data from public sources that do not need user accounts.

**Can I stop a download in progress?**
Yes. You can click the Pause or Stop button at any time. The software saves your progress.

**Where does the software save my files?**
The default location is in your Documents folder under a subdirectory named Schola-Herv. You can change this in the preferences window.

**What if a download fails?**
Check your internet connection first. If the problem persists, the software will attempt to re-download the file automatically during your next session.

**Does this work for all academic papers?**
The tool focuses on open-access content. If a paper is behind a paywall, the tool may not be able to retrieve the full PDF file.

## 📁 Data Management Tips

Keep your folders organized as you collect data. Rename files if you need to track specific authors or publication years. The tool generates a CSV file with your search metadata. This file helps you track the history of your research project. 

If you plan to build a large machine learning corpus, verify the total disk space before you start a long-running download process. Academic datasets grow quickly. Clear out unwanted files regularly to maintain system performance.
